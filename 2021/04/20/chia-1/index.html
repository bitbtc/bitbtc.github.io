什么是chia？

Chia(奇亚)于2017年8月注册成立，旨在开发一个改进的区块链和智能交易平台。我们正在构建Chia网络，以改善全球金融和支付系统。Chia将成为第一个企业级数字货币。Chia使用的是比特币以来第一个新的中本聪共识算法。它被称为 “空间和时间证明”（Proof of Space and Time），是由目前最优秀的网络协议工程师、BitTorrent的发明者Bram Cohen创建的。Chialisp是Chia新推出的智能交易编程语言，功能强大、易于审核、安全。目前可供参考的智能交易有：原子交换、授权收款人、可恢复钱包、多重签名钱包和限价钱包。

和比特币、FIL区别

和传统挖矿的区别
与BTC、ETH采用工作量证明（POW）不同的是，Chia(奇亚)采用”空间和时间证明”（Proof of Space and Time），利用存储设备（电脑、服务器、NAS、树莓派）中的剩余存储空间进行挖矿。

在开始挖矿之前需要在闲置硬盘空间进行播种（plots），然后农民（farmer）在已经播种好的P盘文件（plots files）上寻找最佳爆块答案。Chia硬盘挖矿的本质就是先将加密数据写入硬盘（plots过程），然后挖矿程序在P盘文件中寻找最优答案（farmer过程），所以你的P盘文件总容量相对于全网容量占比越高，越容易爆块，对应的挖矿收益也越高。

播种的过程需要占用CPU、内存（每个P盘线程默认需要4G内存空间）和硬盘临时空间（生成K = 32规格的播种文件大概需要占用332G临时空间）。因此播种（Plots）的过程建议采用性能较好的CPU、16G以上内存和固态硬盘（建议合计2T以上的M2.NVME协议SSD）。参考Plots平台硬件配置表

Chia与Filecoin的区别

咨询中，我发现很多朋友混淆了Chia和Filecoin，二者都是使用硬盘挖矿，但是却有本质区别。

Filecoin其实是IPFS上的激励层，主打存储和带宽分享，硬盘用于封装存储好的用户数据，你可以把它想像成一个去中心化的云盘。

Chia并不提供存储服务，硬盘中存储的数据是完全无意义的，是为了存储而存储，从而证明你真的有这么大的硬盘空间。

Filecoin有很高的挖矿门槛，想挖矿，你需要先购买FIL做抵押，如果你的矿机在线率不达标，抵押的FIL就会被部分或全部没收，真的可以做到负收益。

这样设计主要是考虑到服务可用性，你如果是用户，用FIL购买了存储服务，你也不希望等你想用数据时，存储你数据的机器不在线吧。

因此，Filecoin需要使用这种抵押机制，逼迫矿工优化技术方案，保证长时间在线。

同时，Filecoin想要存储用户的数据，必须通过网络先把用户数据下载下来，等到用户需要时，再上传给用户，这需要大量的带宽资源，显然这不是普通家用宽带能满足的。

与Filecoin相比，Chia的门槛更低，设备上，硬盘无论是功耗、成本、货源都远比显卡有优势，这就是为什么近期它受到大家追捧的原因。

什么是时空证明？

空间证明是一种加密技术，证明人会证明他们将未使用的硬盘驱动器空间分配给存储空间。不过，如果要将「空间证明」用作为一种共识机制，就必须将其与时间证明联系起来，时间证明可以确保区块之间的时间具有一致性，继而提高了区块链的整体安全性。

空间证明和时间证明如何工作？

空间证明，可以视为一种「证明你在硬盘驱动器上保留了一些未使用存储」的方式。Chia 区块链用户需要通过安装软件来快照自己在硬盘驱动器上未使用的空间，这种操作也被称为「播种」（seed），这个软件会在磁盘上一组加密数字存储到「地块」（plot）中，而执行这些操作的人被称为「农民」（farmer）。当区块链广播下一个区块挑战时，农民可以扫描他们的「地块」以查找是否具有最接近挑战的哈希值，Chia 会根据农民所拥有的空间大小和整个网络空间大小比例分配赢得一个区块的概率。

时间证明要求区块之间传递「必须」有一小段时间。时间证明是通过可验证延迟函数（VDF：Verifiable Delay Function）实现的，该函数需要花费一定的时间来计算，但验证速度其实非常快。可验证延迟函数的核心思想是顺序计算，由于大量并行设备不会产生任何收益，因此可以最大程度地减少电力浪费。VDF 服务器（「Timelords」）数量可能会相对较少，因为验证速度最快的 VDF 服务器将会始终排在最前面，并且网络上仅需要一个快速且公平的 Timelord 即可完成一个区块并将整条区块链向前推进。

Chia 优势何在？

Chia 采用全新的中本聪共识算法，该算法可以彻底消除区块链系统工作量证明的能耗需求。与其他加密货币相比，Chia 区块链更加去中心化，安全性也得到大幅提高。Chialisp 是 Chia 区块链的全新智能交易编程语言，它功能强大、易于审核且十分安全，能够更好地释放加密货币所承诺的安全性、透明性和易用性。

此外，Chia 还采用了更先进的加密工具来实现丰富的智能交易功能。通过一家公开、开源、营利性开发公司，Chia 正在采用一种全新的高级解决方案来构建和支持区块链，并向区块提供支持。Chia 将使用「预耕作 pre-farm」（战略储备）来解决代币波动性问题，旨在减轻市场泡沫或市场崩盘对 Chia 带来的影响，同时进一步推动 Chia 采用。

为什么要进行「预耕作」？

Chia 具有一套新颖的商业模式，既可以降低代币波动性，又可以提高采用率。通过向 Chia 借贷并管理这些贷款利率以及其他工具（例如，使用 Chia 代币购买我们的股票），我们希望代币季度波动率可以得到进一步降低。

为了进一步推动应用普及，我们打算把 Chia 借给全球 5000 家公司，这些公司将使用 Chia 向全球供应商提供快速、廉价、安全地付款。

我们还打算利用战略储备来协助开发和采用工作，还有其他一些事情，比如在 Chia 生态系统中投资一些有前途的初创公司，并在有限时间内增加耕作奖励以刺激其他耕作活动。如果企业使用 Chia 而非法定货币进行国际付款，那么可以获得 105% 转账金额「福利」。

Chia 会进行 ICO 吗？

不，Chia 没有 ICO 计划。相反，我们的目标是在美国证券交易所公开上市。这样，股东可以透明和公开的方式与管理层共享风险和回报，我们还可以利用充分理解的公司控制权，就 Chia Network 打算如何使用战略储备做出有约束力的声明。Chia 数字货币是一种有用的付款工具，但不是投资机会。Chia 打算完成符合监管规定的、通过 SEC 注册的 IPO 上市，并将在主网启动择机上市。

Chia 是否使用了比特币代码？

没有。Chia 代码都是自己原创的。

Chia 的价值会像其他加密货币一样波动吗？

我们相信，由于我们的计划是上市，因此代币波动性将比其他加密货币更小，这将使金融机构可以利用代币和股票实现对冲。

最初，我们预计 Chia Network 的估值将主要基于公司将在其资产负债表资产负债表上对 Chia 的估值，Chia 价格在数字交易平台上的波动很可能反映在证券交易所的股票价格变动上。与传统的区块链项目相比，有更多的方法可以让人们了解 Chia 的成功，这也将使我们股票的常规期权和衍生产品可用作类似 Chia 代币价格的合成衍生产品。此外，我们使用 Chia 战略储备的能力也可能有助于减少市场中 Chia 的波动性。

chia常见问题解答

#问题1：生成PLOT文件的选项中，队列与并发有什么区别？
答：队列是指一个完成生，再自动生成下一个。并发是指同时生成多个PLOT文件，但要考虑SSD缓存区是否够大，CPU 和内存是否跟得上。当然是并发越多越好的。

问题2：一个PLOT文件完成生，缓存区需要手动删除么？
答：不需要，系统会自动删除，并把生成的PLOT文件复制到最终目录。

问题3：挖CHCA需要样的硬件？
答：可参考这篇文章：https://blog.csdn.net/shidaoyu/article/details/115795262。

问题4：临时目录不用SSD可以么？
答：可以。但是临时目录的速度决定了PLOT文件生成的速度，所以建议用高速SSD。而且机械硬盘并发P盘有可能会卡死。

问题5：机械硬盘太贵了，用移动硬盘可以么？
答：可以是可以，但USB口速度不快，稳定性也一般。可以考虑把里面的硬盘拆出来接到主板上
国内的硬盘全部疯涨了，可以考虑亚马逊海外购，还没涨太多，就是到货时间长些。如这个东芝的10T企业盘，2000左右到手。Toshiba 东芝 10TB SATA 512e 7200RPM 3.5 英寸企业硬盘 - MG06ACA10TE

问题6：我有两个SSD，怎么合理运用来并发P盘？
答：可以组RAID，最简单的就是 选择临时目录处，点显示高级选项，则可以选择第二临时目录。另外也可以分两次以上来生成PLOT，如你想同时P4个，先选择并发1、2个，临时目录选第1个SSD，确定后再去选择并发生成第3、4个，临时目录选择第2个SSD，具体个数要根据SSD的剩余空间来定。
问题7：用钱包挖和用矿池挖有什么区别？
答：用钱包挖收成只会是整数的币，如1只这样，可能好久都没有收成。矿池则是和众多网友分成，一天收入0.00000x只币这样，另外矿池是要收手续费的，一般是5%以上。

问题9：绘图过程中可以重启电脑或关闭软件么？
答：可以，但之前的绘图进度不会保存，只能重新开始。

问题10：临时目录和最终目录可以用中文名么？
答：最好不要用中文，否则可能有各种问题。建议这两个目录及上层目录都不要有中文 和特殊字符。

问题11：为什么我生成了PLOT后，几天都没有收成？
答：钱包软件中会显示每次收成的预估周期，PLOT越多，周期越短。但并不是说这个周期就一定有收成。所以看着那个 0 感觉挖了个寂寞。不过相信随着你的PLOT文件越来越多，总是会有收成的。

问题12：当机械硬盘P满后，固态还有用么？
答：这时固态可以不用了，因为固态主要是为了P盘速度，挂机挖矿时是不需要的。但为了更多收益，很多人会继续加硬盘来P，所以固态基本是春蚕到死丝方尽，蜡炬成灰泪始干的了。

问题13：当机械硬盘P满后，PLOT文件是可以删除了么？
答：喀，这个肯定是不能删除的，因为挖矿就是靠这些PLOT文件的。至于挖到矿的PLOT文件是否可以删除，博主还没挖到过，有待考究：）

问题14：为什么我安装了钱包软件后，桌面多了一些其它不是我安装的软件图标？
答：估计你钱包软件不是在官方网站下载的，所有钱包、矿池软件一定要在对应的官方网站下载，不然可能会下载到有木马、广告的软件，造成损失！CHIA的官网网址为：https://www.chia.net/
————————————————
问题16：必须要联网状态下才能生成PLOT文件么？
答：不需要。绘图可以完全脱机完成，不需要区块链完成，但耕种时是需要联网的。

问题17：选择新建PLOT时，“桶”的设置有什么用？
答：“桶”默认设置是128个，是指使用的存储数量，如果增加了桶的数量，使用的内存则会减少，但会生成更多的临时文件和磁盘入量。

问题18：做好的PLOT文件可以移动到别的目录或者硬盘中么？
答：可以。PLOT只是一个静态文件，可以移动到别的目录、硬盘甚至是另一台计算机中（不同操作系统也可以）。但如果虽移动到有收割机在运行的计算机，建议将PLOT文件从* .plot重命名为* .plot-mv，移动好后再改回*.plot，这样收割机就不会认为这个是损坏文件了。

问题19：生成PLOT出错或者是电脑不小心重启了，缓存目录中的文件需要手动删除么？
答：需要手动删除的，中途出错、重启等情况下，缓存目录中的文件夹不会自动删除，即使你新开一个PLOT文件，之前的临时文件仍然存在，可能会导致空间不足。
