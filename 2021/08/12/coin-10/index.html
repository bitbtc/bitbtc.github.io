NFT进入主流市场仍面临5大挑战

原文标题：《The NFT space is booming again, but experts say there are 5 challenges the digital collectibles space still needs to overcome》
原文来源：markets.businessinsider
原文编译：0x33，律动BlockBeats

2021 年，NFT 的人气猛增，上半年销售额达到令人瞠目的 25 亿美元，流行歌手 Katy Perry 到说唱歌手 Jay-Z 等一系列名人，以及 NBA 以及 LV 等机构，都纷纷进军 NFT 市场。

区块链公司 CurrencyWorks 创始人兼董事长 Cameron Chell 表示，尽管 NFT 正在经历指数级增长，但该行业仍处于采用曲线的早期阶段。Chell 表示:「收集是人类心理的一部分。「随着区块链的出现和使用 NFT 来创建真正稀缺的数字资产，收集我们认同的东西的继承行为现在可能会发生。」

NFT 是绑定在区块链上的任何形式艺术品的数字表示。每个 NFT 都有一个签名，可以在公共分类帐中核实，不能被复制或编辑。当人们购买 NFT 时，他们获得的是独特代币的权利，而不是艺术品本身。

然而，即使在经历了突破性的一年之后，该行业仍处于起步阶段。专家认为，NFT 市场要想进一步进入主流市场，需要做以下五件事。

  1. 真实性

Pastel 的总裁兼首席运营官 Anthony Georgiades 说，目前确定 NFT 真实性的工具有限。Pastel 使用的工具不仅通过搜索自己的市场，也通过搜索扫描其他市场来验证 NFT 的真实性。进一步保证真实性对收藏者来说很重要，特别是在欺诈很常见的加密市场中。

  1. 可扩展性

目前，大多数 NFT 交易市场是由 Layer 1 区块链驱动的，比如以太坊。Georgiades 说，为了以高度可扩展的方式支持 NFT 的特定功能，需要制定一个专门建造网络来支持 NFT 的发展。可扩展性是 NFT 被主流采用过程中最大的缺陷之一。

  1. 互操作性

虽然大多数 NFT 都在以太坊区块链上，但其他区块链上同样也在发展自己的 NFT 市场，比如 BSC 以及 Flow。这会导致一些问题出现，因为 NFT 拍卖平台 Cryptograph 的联合创始人 Hugo McDonaugh 表示，NFT 位于不同「城市」，不允许它们共享可扩展性、安全性和消息传递会造成 NFT 之间流动性阻塞。Hugo McDonaugh 解释到，这就是为什么他正在帮助波卡网络的发展，他称之为「所有这些城市之间的道路」。

  1. 存储

获取 NFT 是一回事，但是底层数据的存储就不是那么简单了。大多数 NFT 只是具有指向中心化云存储解决方案 (如谷歌 cloud 和 Amazon Web Services) 的外部超链接。Pastel 的总裁 Georgiades 说，这样做会带来很多风险，让任何创造者或所有者都非常容易遭受资产损失，这些资产可能会在所有者不控制的情况下消失、改变或下线。

点对点存储系统 IPFS 正在获得越来越多的关注，但可能会有些难以处理，因为它需要个人持有者维护世界各地的节点。

  1. 可访问性

McDonaugh 称，越来越多的投资者可能开始意识到什么是 nft，但参与门槛仍然相对较高。他说，能够降低这一障碍的是一个二级市场，在那里，所有「城市」的所有货币和资产都可以在买卖时被接受。
