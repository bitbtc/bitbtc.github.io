NFT 观点｜区块链如何重塑艺术价值和创意产业结构？

NFT 观点｜区块链如何重塑艺术价值和创意产业结构？

区块链作为一项推动艺术领域快速发展的底层技术，不仅激活了艺术家的策略、可能性及创业模式，而且拥有重塑艺术和创意产业结构的潜力。

区块链底

区块链本质上是一种特殊的数据库结构，是一个有时间戳的信息账本。 区块链的创新之处在于其分布式账本，也就是说，它存在于许多相互关联的副本中，省去了需要中央权威进行委托的环节，无需依靠博物馆、政府、银行等机构作为可委托的保管者。

此外，区块链的分布式账本上还建立了金融结构，奖励和激励人们保持账本副本更新。 通过解决区块链上的问题，用户可以获得加密货币、比特币和其他货币作为回报。因此，不难看出，从根本上说，区块链是一种知识技术。当然，该技术仍在调整中，伴随着诸多风险和挑战。

从方法论的角度来说，很难评估区块链是不是一种静态的技术，既因为它是艺术管理等复杂的跨学科动态的缩影，也因为区块链是一个刚刚崭露头角的新技术，目前没有配套的社会科学研究方法。虽然金融领域已经出现了区块链的实证研究，但艺术领域的调查、统计、分析会倾向于描述过去的采用情况或当下持有的态度。但毋庸置疑的是，这项技术具有让艺术领域产生翻天覆地的变化的潜力。

接下来，本文将会从区块链技术和 NFT 的到来如何赋能艺术，对数字艺术领域创业的影响，加密艺术新平台以及所面临的挑战进行深度解析。

区块链简

虽然很多人认为中本聪的比特币白皮书和比特币区块链的推出引入了区块链技术，但实际上，比特币更早以前便已经出现。 中本聪的比特币白皮书一共有八个脚注。其中有三个脚注由 Stuart Haber 和 Scott Stornetta 撰写，其中提到的区块链时间戳结构早于白皮书发表 20 年。

结合当时的社会发展，Haber 和 Stornetta 提出了两个问题。 第一个问题是哲学问题：如果在电脑操作文件如此之简单，那么我们如何知道过去的真实情况？另一个是政治性问题：我们如何才能相信我们所知道的过去，免去依赖中央机构保存记录？

事实证明，在没有中央管理员的情况下建立一个值得信赖的数字文件登记系统相当困难，Haber 和 Stornetta 几近放弃。

后来，Stornetta 灵感的迸发让两人开始着手建立区块链系统。两人所想的时间戳账本——区块链的基本底层结构——既有关密码学，又有关记录学。带有时间戳的记录连接在一条链条上，在不破坏整个链条的情况下，无法人为篡改其中任何一项。

后来，中本聪采用了分布式账本概念，加入了维护账本副本连接的经济激励。中本聪的关键之举是发明了“挖矿”——通过解决和验证区块中的交易问题来赢得比特币。

虽然区块链的功能远远超出了普通货币，但比特币在发明初期的轨迹远远超出人们想象。

比特币推出约 15 个月后，有人试图强行用比特币购买实体产品。一位名叫 Laszlo Hanyecz 的计算机程序员向任何愿意为他采购披萨的人提供比特币。几个月后，与他进行交易的英国人换来的比特币价值百万。

中本聪引入挖矿机制后，迎来了一小波加密货币浪潮。2014 年，Vitalik Buterin 引入了以太坊协议——一种允许代币化的智能合约结构。 与区块链原始协议比特币不同（需要用户掌握更多功能机制），以太坊推出了更简单的界面，并引入了代币结构，行使金融功能。以太坊还使得比特币的一些脚本语言普遍化，让其能在多种类型的程序上运行。

随着时间的推移，其中一些程序——智能合约——已经变得标准化，如 ERC-20 （同质化代币协议）和 ERC-721 （非同质化代币协议）。 而非同质化代币正是数字收藏品的基础，如 Cryptokitties 等，使得每一幅艺术作品都能体现其独一无二的特点。

虽然区块链技术还在不断发展中，但它有可能和其他引发社会大规模变革的技术革命一样，产生深远影响。在不远的未来，区块链的发展可能会在相对较短的时间内，对艺术史学家、艺术家、保护者、收藏家、经销商、博物馆以及文化资产和创意产业生态系统产生深远影响。

加密艺术

加密艺术是什么？为什么能引起颠覆和狂潮呢？

加密艺术最精确的定义是：通过区块链非同质化代币产生稀缺性的数字艺术。艺术品只有在稀缺的情况下才有价值。 绘画和雕塑本质上是稀缺的。照片、蚀刻版画或石版画通常可戴上艺术家签名，成为限量版。但数字文件可以无限复制，远不可能成为稀缺——这是在区块链发明之前的情况。

区块链的诞生旨在保证数字货币的稀缺性，省去信托中央机构的流程。区块链上的代币也可以代表金钱之外的其他东西，比如艺术品。

然而，金钱和艺术品相去甚远。金钱是同质化的，就像纸币一样，不看单位，只看面额。比特币也是如此。每个代表价值的代币都和其他代币功能一样，具有统一性。

区块链如何重塑数字艺术？

区块链在艺术领域的核心应用包括出处验证、真实性记录、新媒体和生成性艺术的数字稀缺性、碎片化所有权和共享所有权以及新形式的版权记录。 基于以太坊的智能合约也带来了更多投资选择——引入了创新的知识产权结构。

在传统艺术中，为筹备艺术展览，画廊需要耗费大量时间进行作品出处研究。审查成本包含在佣金等画廊加价收费结构中。 但区块链技术带来的远不止成本的降低，其对于出处的验明甚至可以决定作品是否能够成功出售。

未来，区块链可以解决缺乏文书记录和需要所有权证明的问题，充分利用不可更改和去中心化记录的潜在力量。 如果区块链数据库成为产权登记处，可以证明合法所有权，那么作品的合法所有权就离不开区块链的出处证明。

这也就意味着，没有区块链记录转移，艺术品的所有权也就不能转移。甚至如果市场希望作品“链下”出售时，下一级市场有权决定是否承认其作品出售权利。

最关键的是，区块链记录不仅可以起到证明的作用，还可以作为一个值得信赖的的源头资料库。区块链记录不一定要即刻证明作品真假，也可以包括人类专家审核的信息。

比如，2015 年成立的 Verisart 公司便是为艺术作品证明提供可信赖的数据库；以太坊公司 Artory 为收藏家提供区块链加密的权威性证书；Codex Protocol 使用 BidDex 代币来存储藏家发布的信息。

Verisart、Artory 和 Codex 现在都面临着管理“区块链气隙（blockchain air gap）”的问题，即区块链作品和实物艺术品之间的区别。 为了将两者联系起来，各家公司已经开始进行多种探索，包括从物理标签、DNA 分析、指纹的方式记录等。某些区块链数字艺术作品应用可能能够解决这些问题。

区块链处理的数字艺术交易的另一核心挑战是如何创建可复制作品的限量版本。 艺术界是一个重视新奇、稀缺、独创的领域，与作品的价值息息相关，而复制粘贴显然正在产生破坏。

除了通过版本实现数字稀缺性外，区块链还允许单个艺术品的零碎或共享所有权。 从收藏家的角度来看，将单个艺术品所有权零碎化可以使作品持有方式多元化。从艺术家的角度来看，出售代币或保留股权可以达到获得转售版权费用效果，这跟传统艺术品交易市场大为不同。

虽然这些区块链应用将会提供新方式来证明版权，体现创作的合作性质，但本质上这些应用仍是投机性的；同时，这些应用将剧烈颠覆某些投资和权利管理方式。

NFT 的到来

艺术品是独一无二的。这也是 NFT 之所以被创造的原因。每个 NFT 都各不相同，是非同质化的。 它有独特身份，且带有元数据（图像、描述等）。NFT 可以通过互联网发送，存储在钱包中，这一点跟比特币一样。

非同质化即指某一物品的独一无二性，如一件原创艺术品、一份餐馆餐食。 非同质化代币是 2017 年 9 月发布在以太坊区块链上的技术协议，允许创建可认证的单一数字资产。每个 NFT 都与不同的标识符绑定，使得每个代币不可互换，对其所有者来说是独一无二的。开发者可以创建一个合约，其中每个代币都拥有不同的价值，因此用户可以创建一个可以证明为自己所有的数字资产，出售换取真实利益。

NFT 可以代表任何类型的收藏品，数字艺术已经成为其最受欢迎和最成功的使用案例。 加密艺术最大的颠覆性在于为创作者提供更好的经济效益。在加密平台中，省去了中间环节，实体艺术品的创作者可从画廊销售中获得约一半的收益，超稀有作品术家更是能从初级销售收入获取 85% 的利益。 艺术家的经济效益明显优于传统系统。对于新兴艺术家，这种初次销售收入多、二次销售经济效益潜力大的平台更是有利于他们站稳脚跟。

传统绘画也可以用 NFT 来代表。这种情况下，NFT 的最大优点不是稀缺性，而是区块链代币可以证明艺术品的真实性和出处。

传统市场中，收藏家须得相信画廊证明艺术品真伪。画廊须出示文件，显示艺术品信息。而区块链技术提供了更高效的办法。这无疑提高了艺术家的地位。

区块链对艺术领域创业的影响

区块链对艺术创业主要有三大影响。

营利性和非营利性

首先，区块链将非营利的公共功能引入了私立初创公司，模糊了艺术组织中营利性和非营利性的界限。这反映了基础设施尚不存在时，新技术往往会创造新商机，而新公司往往需要创造基础设施。

现在所需的基础设施功能包括出处研究和产权登记。在房地产等其他行业，这种功能是由政府机构完成。但加密艺术领域也并非如此。Artory、Verisart 和 Codex Protocol 等开拓性区块链公司正在全力建设这些功能，有望成为艺术品所有权方面值得信赖的登记机构。

此外，区块链从根本上说是去中心化的，也就导致基础设施的问题更加复杂。现有的记录保存基础设施是由中心化平台——如政府、博物馆、权利管理机构——来操作的，而区块链颠覆了中心化机构，主要依赖算法权威和公众意见。

新型资金结构

区块链在艺术创业中的第二大潜在影响领域是关于机构和政府的新型资金结构。理论上，我们可以将剧院组织比喻为股份公司。区块链剧院创建的投资结构中，顾客可以购买代币并拥有剧院的一部分，而不是只买票或慈善捐赠。这些模式从消费模式——买票——转变为占有模式——购买股份的投资模式。这些投资模式也可以适用于非营利性艺术组织和政府部门。

艺术金融化新结构

区块链对艺术创业的第三个关键影响在于艺术金融化的新结构。虽然艺术品拥有许多社会和文化价值，不能直接还原为金融价值，但区块链的发展方向有可能从根本上改变艺术品在市场上的流通方式。

例如，Cryptokitty 的非同质化代币是一个独一无二的形象，但同时也是一系列中的一个作品。尤其要考虑基于以太坊的 ERC-721 NFT 的金融功能大体和艺术作品类似。Portion、Otis 等初创公司不仅允许投资艺术品，也允许投资运动鞋或其他收藏品，这些公司可能会改变我们只把艺术品看作资产的想法。

随着众多公司开始建立艺术和文化资产投资平台，艺术领域两个基本方面将发生变化。

第一，艺术市场氛围的转变。 这种情况会使艺术投资的经济和文化符号发生转变，并将艺术投资与奢侈品收藏紧密联系起来。虽然共享投资结构可以解决艺术投资中多元化和流动性问题，但也有可能破坏艺术市场的稳定。

第二，碎片化艺术投资也可以将艺术市场重新定位到艺术家上，让艺术家本身受到关注。

面临挑战

除此之外，区块链重塑艺术之路还面临着众多挑战，包括实体艺术品和政府监管带来的问题。

从实体艺术品来看，目前，还没有一种标准方式能将区块链记录与实体艺术品联系起来。

从监管风险来看，各国政府一直在更新对于加密货币的监管方式，使其融入现有税收和监管体系。

加密市场新机遇

但同时，为了满足艺术家和收藏家的需求，几个市场应运而生，更多市场即将推出，为加密艺术产业发展增添新动力。

包括不限于：

OpenSea.io

每个人都可以上传自己的艺术作品，以数字货币的形式出售。

SuperRare.com

艺术家须得经过申请，平台只接受独一无二的艺术品。

Rarible.com

对任何人开放，平台提供了一种创建 NFT 的简单方式。

这些平台大多基于以太坊，但 WAX、RSK 等也开始进入市场，分一杯羹。这些市场平台为艺术家提供的另一个优势是，艺术家可从二次销售中获得份额，通常在 10% 左右。

在传统艺术行业中，一旦艺术家将其作品卖给收藏家，如果收藏家以更高的价格转卖，他们获不到任何利润。在加密市场中，艺术家得到了他们应得的份额。除经济利益之外，加密技术还带来了新表达形式。许多加密艺术作品不是静态的，而是动画的。还有些是互动的，带有计算机代码。

未来发展

区块链对于问题解决的速度令很多技术都望其项背。除了对艺术市场的颠覆以外，区块链技术还可能用艺术开启其他投资方式。

非同质化代币可以让收藏品投资变得像艺术品投资一样。如果 Cryptokitties 能够作为艺术作品让更多人接受，那么博物馆和其他面向公众的艺术实体作品可能会颠覆大众对于艺术价值的认知。当这些作品进入市场时，都会从这一领域的领跑者身上吸取经验，获得艺术新生。

随着区块链数字环境的不断发展，加密艺术未来也将呈现更多可能性。

比如，艺术展示将会与虚拟现实结合。 传统市场中，受物理空间限制，博物馆只展示 5% 的藏品，顶级的艺术家的展示也不过一半。部分问题在于展示所需的物理空间。但虚拟世界则没有限制。Async Art 的专用数字框架就是很好的例证。

未来沉浸式的虚拟环境也会与加密艺术相融合，提供新的看待艺术的方式，创造新的与艺术互动的形式。未来，艺术创意作品可能会向混合媒体 NFT 的方向扩展，用视觉、声音和多种创意表达来描绘世界。
